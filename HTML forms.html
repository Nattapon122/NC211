4.3.The input maxlength attribute

แอตทริบิวต์ maxlength ระบุจำนวนอักขระสูงสุดที่อนุญาตในช่องอินพุต:






4.4.The input multiple attributes

แอตทริบิวต์หลายรายการระบุว่าผู้ใช้ได้รับอนุญาตให้ป้อนมากกว่าหนึ่งค่าในช่องป้อนข้อมูล.



ในการเลือกหลายไฟล์ให้กดแป้น CTRL หรือ SHIFT ค้างไว้ขณะเลือก

4.5.The input placeholder attribute

แอตทริบิวต์ตัวยึดระบุคำใบ้สั้น ๆ ที่อธิบายค่าที่คาดหวังของช่องป้อนข้อมูล.



4.6.The input required attribute

แอตทริบิวต์ที่จำเป็นระบุว่าต้องกรอกข้อมูลในช่องป้อนข้อมูลก่อนส่งแบบฟอร์ม

4.7.The input step attribute

แอ็ตทริบิวต์ step ระบุช่วงเวลาตามกฎหมายสำหรับองค์ประกอบอินพุต.

4.8.The input height and width attributes

แอตทริบิวต์ความสูงและความกว้างระบุความสูงและความกว้างขององค์ประกอบประเภทอินพุต = "รูปภาพ".





Note: ประเภทอินพุต = "รูปภาพ" จะส่งพิกัด X และ Y ของการคลิกที่เปิดใช้งานปุ่มรูปภาพ.

4.9.The input list attribute

แอตทริบิวต์รายการหมายถึงองค์ประกอบดาต้าลิสต์ที่มีตัวเลือกที่กำหนดไว้ล่วงหน้าสำหรับองค์ประกอบอินพุต.

5.The input form attribute

แอตทริบิวต์ form ระบุรูปแบบที่องค์ประกอบอินพุตเป็นของ.



5.1.The input formaction attribute

แอตทริบิวต์ formaction ระบุ URL ของไฟล์ที่จะประมวลผลอินพุตเมื่อส่งแบบฟอร์ม.





5.2.The input formenctype attribute

แอตทริบิวต์ formenctype ระบุวิธีการเข้ารหัสข้อมูลแบบฟอร์มเมื่อส่ง.



5.3.The input formmethod Attribute

แอตทริบิวต์ formmethod กำหนดเมธอด HTTP สำหรับการส่งข้อมูลแบบฟอร์มไปยัง URL การดำเนินการ





5.4.The input formtarget attribute

แอตทริบิวต์ formtarget ระบุชื่อหรือคีย์เวิร์ดที่ระบุตำแหน่งที่จะแสดงการตอบกลับที่ได้รับหลังจากส่งแบบฟอร์ม.
